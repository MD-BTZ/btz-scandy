Was ist Scandy?

Scandy ist ein webbasiertes Verwaltungssystem für Bildungseinrichtungen, Werkstätten und Betriebe. Das System ermöglicht die Verwaltung von Werkzeugen, Verbrauchsmaterial, Mitarbeitern und Arbeitsaufträgen.

Schritt 1: Stammdaten pflegen (Admin-Bereich)

Bevor Sie mit der Nutzung beginnen, müssen einige grundlegende Einstellungen vorgenommen werden.

Kategorien verwalten

Erstellen Sie Kategorien für Ihre Werkzeuge und Materialien (z.B. "Handwerkzeuge", "Elektrowerkzeuge", "Verbrauchsmaterial").

  • Navigieren Sie zum Dashboard.
  • Suchen Sie den Abschnitt "Kategorien".
  • Geben Sie den Namen der neuen Kategorie in das Eingabefeld ein.
  • Klicken Sie auf den Button "Hinzufügen".
  • Zum Löschen auf das Mülleimer-Icon klicken (nur möglich, wenn die Kategorie nicht mehr verwendet wird).

Standorte verwalten

Definieren Sie Lagerorte für Ihre Werkzeuge und Materialien (z.B. "Werkzeugraum", "Lager", "Werkstatt").

  • Navigieren Sie zum Dashboard.
  • Suchen Sie den Abschnitt "Standorte".
  • Geben Sie den Namen des neuen Standorts in das Eingabefeld ein.
  • Klicken Sie auf den Button "Hinzufügen".
  • Zum Löschen auf das Mülleimer-Icon klicken (nur möglich, wenn der Standort nicht mehr verwendet wird).

Abteilungen verwalten

Ordnen Sie Mitarbeiter Abteilungen zu (z.B. "Elektro", "Sanitär", "Verwaltung").

  • Navigieren Sie zum Dashboard.
  • Suchen Sie den Abschnitt "Abteilungen".
  • Geben Sie den Namen der neuen Abteilung in das Eingabefeld ein.
  • Klicken Sie auf den Button "Hinzufügen".
  • Zum Löschen auf das Mülleimer-Icon klicken (nur möglich, wenn die Abteilung keinem Mitarbeiter mehr zugewiesen ist).
Schritt 2: Mitarbeiter, {{ app_labels.tools.name }} & {{ app_labels.consumables.name }} anlegen

Nachdem die Stammdaten gepflegt sind, können Sie die eigentlichen Inventar- und Personendaten erfassen.

Mitarbeiter hinzufügen

Erfassen Sie alle Personen, die Werkzeuge ausleihen oder Material entnehmen sollen.

  • Navigieren Sie zum Menüpunkt Mitarbeiter.
  • Klicken Sie auf den Button "Neuer Mitarbeiter".
  • Füllen Sie die Felder aus:
    • Vorname & Nachname: Selbsterklärend.
    • Barcode: Wichtig! Scannen Sie den Barcode vom Mitarbeiterausweis oder geben Sie die Nummer manuell ein. Dieser Barcode wird beim Scannen verwendet. Er muss eindeutig sein.
    • Abteilung: Wählen Sie eine der zuvor angelegten Abteilungen aus.
    • Email (Optional): Für Benachrichtigungen.
  • Klicken Sie auf Hinzufügen.

{{ app_labels.tools.name }} hinzufügen

Erfassen Sie alle {{ app_labels.tools.name }}, die ausgeliehen werden können.

  • Navigieren Sie zum Menüpunkt {{ app_labels.tools.name }}.
  • Klicken Sie auf den Button "{{ app_labels.tools.name }} hinzufügen".
  • Füllen Sie die Felder aus:
    • Name: Eindeutige Bezeichnung (z.B. "Bohrmaschine Bosch GSB 18V").
    • Barcode: Wichtig! Scannen oder geben Sie den Barcode ein, der am {{ app_labels.tools.name }} angebracht ist. Dieser Barcode dient zur Identifikation. Er muss eindeutig sein.
    • Kategorie: Wählen Sie eine passende, zuvor angelegte Kategorie.
    • Standort: Wählen Sie den Lagerort aus.
    • Status: Wählen Sie "Verfügbar" oder "Defekt".
    • Beschreibung (Optional): Weitere Details.
  • Klicken Sie auf Hinzufügen.

{{ app_labels.consumables.name }} hinzufügen

Erfassen Sie alle {{ app_labels.consumables.name }}.

  • Navigieren Sie zum Menüpunkt {{ app_labels.consumables.name }}.
  • Klicken Sie auf den Button "{{ app_labels.consumables.name }} hinzufügen".
  • Füllen Sie die Felder aus:
    • Name: Eindeutige Bezeichnung (z.B. "Schrauben Senkkopf 4x40mm").
    • Barcode: Wichtig! Scannen oder geben Sie den Barcode ein, der auf der Verpackung oder dem Regalplatz angebracht ist. Dieser Barcode dient zur Identifikation. Er muss eindeutig sein.
    • Kategorie: Wählen Sie eine passende, zuvor angelegte Kategorie.
    • Standort: Wählen Sie den Lagerort aus.
    • Menge: Der aktuelle Lagerbestand.
    • Mindestmenge: Wird dieser Wert unterschritten, erscheint eine Warnung im Dashboard.
    • Beschreibung (Optional): Weitere Details.
  • Klicken Sie auf Hinzufügen.
Schritt 3: Scannen & Manuelle Vorgänge

Nach der Systemeinrichtung können die Hauptprozesse durchgeführt werden.

Quick-Scan

Ausleihen, Rückgaben und Materialentnahmen über Barcode-Scanner.

  1. Öffnen Sie Quick-Scan über die Navigation.
  2. Das System ist bereit für den ersten Scan.
  3. {{ app_labels.tools.name }} Ausleihe/Rückgabe:
    • Scannen Sie den Barcode des {{ app_labels.tools.name }}.
    • Das System zeigt Details an und prüft den Status.
    • Klicken Sie auf "Ausleihen/Zurückgeben".
    • Bei Ausleihe: Scannen Sie den Barcode des Mitarbeiterausweises.
    • Die Aktion wird gespeichert.
  4. {{ app_labels.consumables.name }} Entnahme:
    • Scannen Sie den Barcode des {{ app_labels.consumables.name }}.
    • Das System zeigt Details und Bestand an.
    • Klicken Sie auf "{{ app_labels.consumables.name }} nutzen".
    • Geben Sie die Menge ein, die Sie entnehmen möchten.
    • Scannen Sie den Barcode des Mitarbeiterausweises.
    • Die Entnahme wird gespeichert und der Bestand reduziert.

Manuelle Ausleihe / Ausgabe

Alternative, wenn der Scanner nicht funktioniert oder für spezielle Korrekturen.

  • Navigieren Sie zu Admin -> Manuelle Ausleihe/Ausgabe.
  • Wählen Sie die Aktion (Ausleihen, Zurückgeben oder Materialausgabe).
  • Wählen Sie das entsprechende Werkzeug oder Material aus der Liste aus.
  • Wählen Sie den Mitarbeiter aus (nur bei Ausleihe/Materialausgabe nötig).
  • Geben Sie bei Materialausgabe die Menge an.
  • Klicken Sie auf den Button "Aktion durchführen".
  • Eine Übersicht der letzten manuellen Vorgänge und aktuellen Ausleihen wird angezeigt.
Schritt 4: Ticket-System & Aufgabenverwaltung

Das Ticket-System ermöglicht die Erstellung und Verwaltung von Arbeitsaufträgen.

Tickets erstellen

Erstellen Sie neue Arbeitsaufträge.

  • Navigieren Sie zum Menüpunkt Tickets.
  • Klicken Sie auf den Button "Neues Ticket".
  • Füllen Sie die Felder aus:
    • Titel: Kurze Beschreibung der Aufgabe.
    • Beschreibung: Detaillierte Anweisungen und Informationen.
    • Typ: Allgemein, Auftrag oder Materialbedarf.
    • Zugewiesener Mitarbeiter: Wählen Sie den verantwortlichen Mitarbeiter aus.
    • Priorität: Niedrig, Mittel oder Hoch.
    • Fälligkeitsdatum: Wann die Aufgabe erledigt sein soll.
    • Status: Offen, zugewiesen, in Bearbeitung oder erledigt.
  • Klicken Sie auf Erstellen.

Tickets verwalten

Verfolgen Sie den Fortschritt und aktualisieren Sie den Status.

  • Ticket-Liste anzeigen: Über den Menüpunkt Tickets sehen Sie alle Tickets in einer Tabelle.
  • Status ändern: Klicken Sie auf ein Ticket, um Details zu sehen und den Status zu aktualisieren.
  • Tickets bearbeiten: Ändern Sie Titel, Beschreibung, Zuweisung oder Fälligkeitsdatum.
  • Nachrichten hinzufügen: Kommunizieren Sie direkt im Ticket mit Kollegen.
  • Notizen & Kommentare: Ergänzen Sie wichtige Informationen zum Ticket.
  • Medien anhängen: Laden Sie Fotos, Dokumente oder andere Dateien zum Ticket hoch.

Ticket-Historie

Jede Änderung an einem Ticket wird automatisch protokolliert.

  • Automatische Protokollierung: Status-Änderungen, Zuweisungen, Nachrichten, Medien-Uploads und alle anderen Änderungen werden erfasst.
  • Transparenz: Nachverfolgung aller Änderungen mit Zeitstempel und Benutzer.
  • Visuelle Timeline: Darstellung mit Icons und Zeitstempeln.
  • Dokumentation: Für Qualitätsmanagement und Nachverfolgung.

Auftragsdetails & Materiallisten

Für komplexe Aufträge können detaillierte Informationen hinterlegt werden.

  • Auftragsdetails: Zusätzliche Informationen wie Deadline, Fertigstellungstermin und erweiterte Beschreibungen.
  • Materialliste: Benötigte Materialien mit Mengen und Prioritäten definieren.
  • Bereichszuweisung: Tickets verschiedenen Arbeitsbereichen zuordnen.
  • Zeitschätzung: Geschätzte Arbeitszeit für Planung.

Öffentliches Anfrageformular

Mitarbeiter können über ein öffentliches Formular Ticket-Anfragen einreichen.

  • Anfrage einreichen: Über das öffentliche Formular können Ticket-Anfragen ohne Anmeldung eingereicht werden.
  • Anfragefelder: Name, Kontaktdaten, Beschreibung der gewünschten Aufgabe und Priorität.
  • Automatische Erstellung: Eingereichte Anfragen werden automatisch als neue Tickets im System erstellt.
  • Bearbeitung: Die erstellten Tickets können von Mitarbeitern im System bearbeitet und zugewiesen werden.
  • Statusverfolgung: Anfragende können den Status ihrer Anfrage verfolgen.
Schritt 5: Jobbörse

Die Jobbörse ermöglicht die Verwaltung von Stellenausschreibungen.

Stellenausschreibungen erstellen

Erstellen Sie neue Jobangebote.

  • Navigieren Sie zum Menüpunkt Jobbörse.
  • Klicken Sie auf "Neue Stellenausschreibung".
  • Füllen Sie die Job-Details aus:
    • Stellentitel: Bezeichnung der Position.
    • Unternehmen: Arbeitgeber oder Organisation.
    • Standort: Arbeitsort der Stelle.
    • Anstellungsart: Vollzeit, Teilzeit, Praktikum, etc.
    • Branche: Zugehöriger Wirtschaftsbereich.
    • Beschreibung: Stellenbeschreibung und Anforderungen.
    • Kontaktdaten: Bewerbungsinformationen.
  • Veröffentlichen Sie die Ausschreibung.

Stellenausschreibungen verwalten

Bearbeiten und verwalten Sie bestehende Jobangebote.

  • Übersicht: Alle Stellenausschreibungen in einer filterbaren Liste.
  • Status-Verwaltung: Jobs als aktiv, inaktiv oder abgelaufen markieren.
  • Bearbeitung: Job-Details jederzeit anpassen und aktualisieren.
  • Suchfunktion: Jobs nach Titel, Unternehmen, Standort oder Branche filtern.
  • Statistiken: Überblick über Anzahl der Jobs nach Branchen und Status.

Öffentliche Ansicht

Stellensuchende können die Jobbörse einsehen.

  • Suche: Filter nach Standort, Branche, Anstellungsart.
  • Detailansicht: Stellenbeschreibung mit Bewerbungsinformationen.
  • Pagination: Navigation bei vielen Stellenanzeigen.
Schritt 6: Wochenberichtssystem

Das Wochenberichtssystem ermöglicht die Erfassung von wöchentlichen Arbeitszeiten und Tätigkeiten.

Wochenbericht erstellen

Erstellen Sie einen neuen Wochenbericht.

  • Navigieren Sie zum Menüpunkt Wochenberichte.
  • Klicken Sie auf den Button "Neuer Wochenbericht".
  • Füllen Sie die Felder aus:
    • Woche: Wählen Sie die Kalenderwoche aus.
    • Jahr: Das entsprechende Jahr.
    • Arbeitszeiten: Erfassen Sie für jeden Tag die Arbeitszeiten (Start, Ende).
    • Tätigkeiten: Beschreiben Sie die durchgeführten Arbeiten pro Tag.
  • Klicken Sie auf Speichern.

Wochenberichte verwalten

Verwalten und bearbeiten Sie bestehende Wochenberichte.

  • Berichte anzeigen: Über den Menüpunkt Wochenberichte sehen Sie alle Ihre Berichte.
  • Automatische Erstellung: Für die aktuelle Woche wird automatisch ein leerer Bericht angelegt.
  • Bericht bearbeiten: Klicken Sie auf einen Bericht, um ihn zu bearbeiten.
  • Status-Übersicht: Sehen Sie auf einen Blick, welche Tage vollständig ausgefüllt sind.
  • Bericht herunterladen: Laden Sie Wochenberichte als Word-Dokument herunter.
  • Benachrichtigungen: Das System zeigt Ihnen fehlende Einträge in der Navigation an.

Berechtigungen

Das Wochenbericht-Feature kann pro Benutzer aktiviert oder deaktiviert werden.

  • Admin-Kontrolle: Administratoren können das Feature für einzelne Benutzer ein- oder ausschalten.
  • Rollenberechtigung: Normalerweise für Mitarbeiter und Teilnehmer verfügbar.
  • Eigene Berichte: Benutzer sehen nur ihre eigenen Wochenberichte.
Schritt 7: Backup-System & Systemverwaltung

Das Backup-System ermöglicht die Datensicherung über die Web-Oberfläche.

Backup-System

Alle Backups werden im ZIP-Format erstellt.

  • ZIP-Format: Alle Backups sind komprimierte ZIP-Archive.
  • Datenbank-Backup: Vollständige MongoDB-Sicherung.
  • Medien-Backup: Optional können alle Medien (Fotos, Dokumente) mit gesichert werden.
  • Konfiguration: Wichtige Systemdateien werden automatisch mit einbezogen.
  • Kompatibilität: Alte JSON-Backups können weiterhin importiert werden.

Backup-Verwaltung

Erstellen, verwalten und wiederherstellen Sie Backups über die Web-Oberfläche.

  • Manuelles Backup: Erstellen Sie jederzeit ein Backup über die Admin-Oberfläche.
  • Automatische Backups: Tägliche automatische Sicherungen im Hintergrund.
  • Backup-Liste: Übersicht aller verfügbaren Backups mit Größe und Erstellungsdatum.
  • Download: Backups direkt über die Web-Oberfläche herunterladen.
  • Wiederherstellen: Backups mit einem Klick wiederherstellen.
  • Upload: Externe Backup-Dateien hochladen und wiederherstellen.
  • Löschung: Alte oder nicht benötigte Backups entfernen.

System-Überwachung

Überwachen Sie die Systemleistung.

  • System-Status: Überprüfung der wichtigsten Systemkomponenten.
  • Backup-Status: Übersicht über automatische Backup-Läufe.
  • Speicherplatz: Monitoring der verfügbaren Festplattenkapazität.
  • Logs: Zugriff auf System- und Anwendungslogs über die Web-Oberfläche.
