Scandy Tutorial

Grundlagen zur Einrichtung und Nutzung

Diese Anleitung führt Sie durch die notwendigen Schritte, um Scandy für die Verwaltung Ihrer Werkzeuge und Verbrauchsmaterialien einzurichten und die Kernfunktionen zu nutzen.

Schritt 1: Stammdaten pflegen (Admin-Bereich)

Bevor Sie mit der Nutzung beginnen, müssen einige grundlegende Einstellungen vorgenommen werden.

Kategorien verwalten

Erstellen Sie Kategorien für Ihre Werkzeuge und Materialien (z.B. "Handwerkzeuge", "Elektrowerkzeuge", "Verbrauchsmaterial").

  • Navigieren Sie zum Dashboard.
  • Suchen Sie den Abschnitt "Kategorien".
  • Geben Sie den Namen der neuen Kategorie in das Eingabefeld ein.
  • Klicken Sie auf den Button "Hinzufügen".
  • Zum Löschen auf das Mülleimer-Icon klicken (nur möglich, wenn die Kategorie nicht mehr verwendet wird).

Standorte verwalten

Definieren Sie Lagerorte für Ihre Werkzeuge und Materialien (z.B. "Werkzeugraum", "Lager", "Werkstatt").

  • Navigieren Sie zum Dashboard.
  • Suchen Sie den Abschnitt "Standorte".
  • Geben Sie den Namen des neuen Standorts in das Eingabefeld ein.
  • Klicken Sie auf den Button "Hinzufügen".
  • Zum Löschen auf das Mülleimer-Icon klicken (nur möglich, wenn der Standort nicht mehr verwendet wird).

Abteilungen verwalten

Ordnen Sie Mitarbeiter Abteilungen zu (z.B. "Elektro", "Sanitär", "Verwaltung").

  • Navigieren Sie zum Dashboard.
  • Suchen Sie den Abschnitt "Abteilungen".
  • Geben Sie den Namen der neuen Abteilung in das Eingabefeld ein.
  • Klicken Sie auf den Button "Hinzufügen".
  • Zum Löschen auf das Mülleimer-Icon klicken (nur möglich, wenn die Abteilung keinem Mitarbeiter mehr zugewiesen ist).
Schritt 2: Mitarbeiter, {{ app_labels.tools.name }} & {{ app_labels.consumables.name }} anlegen

Nachdem die Stammdaten gepflegt sind, können Sie die eigentlichen Inventar- und Personendaten erfassen.

Mitarbeiter hinzufügen

Erfassen Sie alle Personen, die Werkzeuge ausleihen oder Material entnehmen sollen.

  • Navigieren Sie zum Menüpunkt Mitarbeiter.
  • Klicken Sie auf den Button "Neuer Mitarbeiter".
  • Füllen Sie die Felder aus:
    • Vorname & Nachname: Selbsterklärend.
    • Barcode: Sehr wichtig! Scannen Sie den Barcode vom Mitarbeiterausweis oder geben Sie die Nummer manuell ein. Dieser Barcode wird später zum Identifizieren beim Scannen verwendet. Er muss eindeutig sein.
    • Abteilung: Wählen Sie eine der zuvor angelegten Abteilungen aus.
    • Email (Optional): Für Benachrichtigungen (falls implementiert).
  • Klicken Sie auf Hinzufügen.

{{ app_labels.tools.name }} hinzufügen

Erfassen Sie alle {{ app_labels.tools.name }}, die ausgeliehen werden können.

  • Navigieren Sie zum Menüpunkt {{ app_labels.tools.name }}.
  • Klicken Sie auf den Button "{{ app_labels.tools.name }} hinzufügen".
  • Füllen Sie die Felder aus:
    • Name: Eindeutige Bezeichnung (z.B. "Bohrmaschine Bosch GSB 18V").
    • Barcode: Sehr wichtig! Scannen oder geben Sie den Barcode ein, der *am {{ app_labels.tools.name }}* angebracht ist. Dieser Barcode dient zur Identifikation. Er muss eindeutig sein.
    • Kategorie: Wählen Sie eine passende, zuvor angelegte Kategorie.
    • Standort: Wählen Sie den Lagerort aus.
    • Status: Wählen Sie "Verfügbar" oder "Defekt".
    • Beschreibung (Optional): Weitere Details.
  • Klicken Sie auf Hinzufügen.

{{ app_labels.consumables.name }} hinzufügen

Erfassen Sie alle {{ app_labels.consumables.name }}.

  • Navigieren Sie zum Menüpunkt {{ app_labels.consumables.name }}.
  • Klicken Sie auf den Button "{{ app_labels.consumables.name }} hinzufügen".
  • Füllen Sie die Felder aus:
    • Name: Eindeutige Bezeichnung (z.B. "Schrauben Senkkopf 4x40mm").
    • Barcode: Sehr wichtig! Scannen oder geben Sie den Barcode ein, der *auf der Verpackung oder dem Regalplatz* angebracht ist. Dieser Barcode dient zur Identifikation. Er muss eindeutig sein.
    • Kategorie: Wählen Sie eine passende, zuvor angelegte Kategorie.
    • Standort: Wählen Sie den Lagerort aus.
    • Menge: Der aktuelle Lagerbestand.
    • Mindestmenge: Wird dieser Wert unterschritten, erscheint eine Warnung im Dashboard.
    • Beschreibung (Optional): Weitere Details.
  • Klicken Sie auf Hinzufügen.
Schritt 3: Scannen & Manuelle Vorgänge

Nachdem das System eingerichtet ist, können die Hauptprozesse durchgeführt werden.

Quick-Scan

Der schnellste Weg, um Ausleihen, Rückgaben und Materialentnahmen zu erfassen.

  1. Öffnen Sie Quick-Scan über die Navigation oder den prominenten Button.
  2. Das System ist bereit für den ersten Scan.
  3. {{ app_labels.tools.name }} Ausleihe/Rückgabe:
    • Scannen Sie den Barcode des {{ app_labels.tools.name }}.
    • Das System zeigt Details an und prüft den Status.
    • Klicken Sie auf "Ausleihen/Zurückgeben".
    • Wenn es sich um eine Ausleihe handelt: Scannen Sie den Barcode Ihres Mitarbeiterausweises.
    • Fertig! Die Aktion wird gespeichert.
  4. {{ app_labels.consumables.name }} Entnahme:
    • Scannen Sie den Barcode des {{ app_labels.consumables.name }}.
    • Das System zeigt Details und Bestand an.
    • Klicken Sie auf "{{ app_labels.consumables.name }} nutzen".
    • Geben Sie die Menge ein, die Sie entnehmen möchten.
    • Scannen Sie den Barcode Ihres Mitarbeiterausweises.
    • Fertig! Die Entnahme wird gespeichert und der Bestand reduziert.

Manuelle Ausleihe / Ausgabe (Alternative)

Nützlich, wenn der Scanner nicht funktioniert, Barcodes unlesbar sind oder für spezielle Korrekturen.

  • Navigieren Sie zu Admin -> Manuelle Ausleihe/Ausgabe.
  • Wählen Sie die Aktion (Ausleihen, Zurückgeben oder Materialausgabe).
  • Wählen Sie das entsprechende Werkzeug oder Material aus der Liste aus.
  • Wählen Sie den Mitarbeiter aus (nur bei Ausleihe/Materialausgabe nötig).
  • Geben Sie bei Materialausgabe die Menge an.
  • Klicken Sie auf den Button "Aktion durchführen".
  • Eine Übersicht der letzten manuellen Vorgänge und aktuellen Ausleihen wird ebenfalls angezeigt.

Damit sind die grundlegenden Schritte zur Einrichtung und Bedienung von Scandy abgedeckt.

Erkunden Sie die weiteren Menüpunkte für Detailansichten, Verläufe und zusätzliche Admin-Funktionen.
